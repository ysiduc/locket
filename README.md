# 📱 Locket Gold for iOS

Kho lưu trữ cấu hình hỗ trợ kích hoạt tính năng **Locket Gold** trên các thiết bị iOS hoàn toàn miễn phí.

## 📦 Danh sách file cài đặt
* [ysiduc0.pac](https://ysiduc.github.io/locket/ysiduc0.pac) (File tự động cấu hình Proxy)
* [ysiduc2.crt](https://ysiduc.github.io/locket/ysiduc2.crt) (Chứng chỉ bảo mật)
* [LocketGold.mobileconfig](https://ysiduc.github.io/locket/LocketGold.mobileconfig) (File cấu hình hoàn tất)

---

## 🛠 Hướng dẫn cài đặt chi tiết

### 🔹 Bước 1: Cấu hình Proxy cho Wi-Fi
1. Sao chép liên kết này: `https://ysiduc.github.io/locket/ysiduc0.pac`
2. Vào **Cài đặt (Settings)** trên iPhone -> Chọn **Wi-Fi**.
3. Nhấn vào biểu tượng chữ **(i)** bên cạnh mạng Wi-Fi đang kết nối.
4. Kéo xuống dưới cùng, chọn **Định cấu hình proxy (Configure Proxy)** -> Chọn **Tự động (Automatic)**.
5. Dán liên kết đã sao chép vào ô **URL** -> Nhấn **Lưu (Save)**.

### 🔹 Bước 2: Cài đặt và Tin cậy Chứng chỉ (Certificate)
1. Mở **Safari**, truy cập trực tiếp vào link: [Tải ysiduc2.crt](https://ysiduc.github.io/locket/ysiduc2.crt)
2. Nhấn **Cho phép (Allow)** khi có thông báo tải cấu hình về máy.
3. Vào **Cài đặt (Settings)** -> Chọn mục **Đã tải về hồ sơ (Profile Downloaded)** ở ngay đầu -> Nhấn **Cài đặt (Install)**.
4. Tiếp tục vào **Cài đặt (Settings)** -> **Cài đặt chung (General)** -> **Giới thiệu (About)**.
5. Kéo xuống dưới cùng, chọn **Cài đặt tin cậy chứng chỉ (Certificate Trust Settings)**.
6. Tìm chứng chỉ vừa cài và **gạt công tắc sang màu xanh** để kích hoạt tin cậy.

### 🔹 Bước 3: Kiểm tra và Hoàn tất
1. Mở ứng dụng **Locket** lên kiểm tra xem tài khoản đã lên **Gold** chưa.
2. *Nếu chưa lên Gold:* Hãy gỡ cài đặt ứng dụng Locket hiện tại, lên App Store tải lại ứng dụng mới rồi kiểm tra lại.
3. Sau khi Locket đã lên Gold thành công:
   * Mở **Safari**, truy cập link: [Tải LocketGold.mobileconfig](https://ysiduc.github.io/locket/LocketGold.mobileconfig)
   * Tiến hành cài đặt cấu hình tương tự như cách làm ở **Bước 2**.
   * Cuối cùng, vào lại **Cài đặt Wi-Fi** -> **Định cấu hình proxy** -> Chọn **Tắt (Off)** để đưa mạng về trạng thái bình thường.
